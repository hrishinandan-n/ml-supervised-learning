# ml-supervised-learning

This repository contains a variety of **supervised learning projects** that demonstrate the implementation of common machine learning algorithms. It is organized by problem type:

## Regression
Includes projects using:
- Linear Regression
- Polynomial Regression
- Ridge Regression
- Lasso Regression
- Decision Tree Regression
- And more...

## Classification
Includes projects using:
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Trees
- Random Forests
- Support Vector Machines (SVM)
- And more...

## Features
- Well-documented **Python code**  
- **Jupyter notebooks** for easy exploration and learning  
- **Datasets** (or references to datasets)  
- Saved **trained models** (where applicable)  

This repository is designed for **learning, experimentation, and showcasing machine learning skills**.
